Руководство пользователя
========================

Правила игры
------------

Монстродел — карточная игра для 2–4 игроков (от 6 лет, 20–30 минут).

**Цель:** собрать монстра с наибольшим количеством звёздочек ★.

**Начало:** каждый игрок берёт 4 карты. Порядок хода выбирается случайно.

**Ход:**

1. Возьми одну карту с верха колоды (автоматически).
2. Сыграй одну карту из руки:

   - **[ОБ] Облик** — часть тела монстра. Кладётся перед тобой, ★ навсегда.
   - **[ДЕ] Действие** — что-то происходит, карта идёт в сброс.
   - **[УГ] Угроза** — плохое для другого игрока, карта идёт в сброс.

3. Или нажми **[п]** — пас (нельзя пасовать два хода подряд!).

**Конец игры:** когда колода заканчивается или кто-то набирает 15★.
Победитель — игрок с наибольшей страшностью.

Управление
----------

В своей строке ввода нажимай:

========  ================================================
Клавиша   Действие
========  ================================================
1–6       Сыграть карту с таким номером из руки
п         Пас (пропустить ход без игры карты)
т         Показать таблицу очков
?         Показать подробное описание карты
========  ================================================

После каждого хода передай клавиатуру следующему игроку.

Уровни страшности
-----------------

==========  =======================
Звёздочки   Прозвище монстра
==========  =======================
0–3 ★       Пугало из огорода
4–6 ★       Кошмарик
7–10 ★      Жуть
11–14 ★     Ужастик
15+ ★       НЕЧТО НЕВОЗМОЖНОЕ!
==========  =======================

Карты
-----

ОБЛИК (20 карт)
~~~~~~~~~~~~~~~

Части тела монстра. Остаются перед игроком навсегда (пока не украдут).

- **Крылья** ×2 — +2★
- **Металл-рога** ×2 — +3★
- **Щупальца** ×2 — +3★
- **Клыки** ×2 — +2★
- **Ядовитые когти** ×2 — +2★
- **Три глаза** ×2 — +2★
- **Бронечешуя** ×2 — +2★, защищает от Пекла
- **Хвост** ×1 — +1★
- **Жало** ×1 — +1★
- **Горб шипов** ×1 — +2★
- **Двойная голова** ×1 — +3★

ДЕЙСТВИЕ (14 карт)
~~~~~~~~~~~~~~~~~~

Разовые эффекты, после применения идут в сброс.

- **Лазеры из глаз** ×2 — выбери игрока: он теряет 1 облик
- **Телепортация** ×2 — укради 1 облик у любого игрока
- **Охота на ближнего** ×2 — укради облик у соседа
- **Камень усиления** ×2 — следующий облик +2★ дополнительно
- **Эволюция** ×2 — возьми 2 доп. карты, сбрось 1
- **Рёв ужаса** ×1 — все другие сбрасывают 1 карту из руки
- **Монстро-шпион** ×1 — тайно посмотри руку любого игрока
- **Мутация** ×1 — замени свой облик на верхнюю карту колоды
- **Кража из сброса** ×1 — возьми любую карту из сброса

УГРОЗА (6 карт)
~~~~~~~~~~~~~~~

- **Пекло** ×2 — без Бронечешуи все теряют по 1 облику
- **Монстролов** ×2 — забери 1 облик у лидера
- **Экзамен по ОКам** ×1 — все сбрасывают 1 карту из руки
- **Зеркало страха** ×1 — поменяйся обликом с лидером

Запуск
------

.. code-block:: bash

   # Стандартный запуск (русский язык)
   monstromaker

   # Английский язык
   monstromaker --lang en_US

   # Передать имена игроков сразу
   monstromaker --players Адель Кирилл Александр Георгий
